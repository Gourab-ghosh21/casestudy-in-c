
1. Start

2. Input x1, y1, x2, y2

3. Calculate difference in x-coordinates:
dx = x2 − x1

4. Calculate difference in y-coordinates:
dy = y2 − y1

5. Calculate absolute value of dx:
abs_dx = sqrt(dx × dx)

6. Calculate absolute value of dy:
abs_dy = sqrt(dy × dy)

7. Calculate Manhattan distance:
total_distance = abs_dx + abs_dy

8. Display total_distance

9. Stop
